## Project Overview
The **Car Rental Management System** is a full-stack web application designed to efficiently manage car rentals.  

**Users** can:  
- Browse available cars  
- Book cars online  
- Make payments  

**Administrators** can:  
- Manage car inventory  
- Track bookings and payments  
- Generate reports  

---

## Technologies Used

**Frontend:**  
- React.js  
- Tailwind CSS / Bootstrap  
- React Router  
- Axios  

**Backend:**  
- Node.js with Express.js  
- MySQL / MongoDB  
- JWT for authentication  

**Other Tools:**  
- Git & GitHub  
- Postman  

---

## Features

### User Features
- User registration & login  
- Browse cars and view details  
- Book cars for specific dates  
- Make secure payments  
- View booking history  

### Admin Features
- Admin login  
- Add, update, and remove cars  
- View all bookings and payments  
- Generate reports
